Cantidad de movimiento para un sistema de particulas:

La "cantidad de movimiento" para un sistema de partículas en dinámica se refiere a la suma vectorial de las cantidades de movimiento individuales de todas las partículas que componen el sistema. También se conoce como "momento total" o "momento lineal total" del sistema. Esta cantidad se relaciona con el movimiento conjunto del sistema y es importante para analizar el comportamiento global de las partículas en interacción.

Formula de la cantidad de movimiento para un sistema de particulas:

 P = Σ m * v
